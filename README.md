
# 3D-SHAPE

Basic 3D modeling program using Three.js


## Tech Stack

**Three.js**

  
## Installation

Go to root folder, install and run project with npm

```bash
  npm install
  npm run build
  npm run start
```
or if you use yarn, run the following commands

```bash
  yarn
  yarn build
  yarn start
```
