
# 3D-SHAPE

Basic 3D modeling program using Three.js


## Tech Stack

**Three.js**

  
## Installation

Go to root folder, install project with npm

```bash
  npm install
  npm run devstart
```
or if you use yarn, run the following commands

```bash
  yarn install
  yarn devstart
```
